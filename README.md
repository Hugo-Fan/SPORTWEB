 SPORTWEB

一個使用 **Spring Boot + MyBatis + Thymeleaf** 開發的  
個人運動紀錄 + 文章 Blog 系統

此專案主要用來練習 Java 後端開發、資料庫設計與前後端整合。

---

專案簡介

SPORTWEB 是我自行開發的個人網站，

主要功能：
- 記錄每日運動數據
- 撰寫文章/心得 (Blog)
- 後台管理資料 (CRUD)

透過這個專案，我練習：

Spring Boot REST API 開發  
MyBatis SQL 操作  
MVC 三層式架構設計  
前後端整合 (Thymeleaf + JavaScript)

---
